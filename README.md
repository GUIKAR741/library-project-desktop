# Library Project Desktop
Aplicativo Desktop Biblioteca

## Bibliotecas Utilizadas
- Kivy
- Cython: 0.29.9
- Pygments
- Docutils
- PyMySQL
- Passlib

## Como rodar esse projeto

### Instale as dependencias com o comando

```sh
pip install pipenv
pipenv install --dev
```

### Execute o projeto

```sh
python main.py
```
